# CompressTextOrImageByHuffman
哈夫曼编码压缩文本和图片课程设计

# 需求分析    

## 所需数据结构     

* 哈夫曼二叉树    
  压缩数据
* 循环队列    
  构建最优二叉树时使用      

整体采用顺序结构存储        

## 所需算法    

* 哈夫曼树构建算法    
* 哈夫曼编码算法    
* 快速排序算法

## 所需API    

* C/C++ IO    
  用于文件、图片数据的读取和保存   
* Opencv HighGUI 
  用于实现用户界面和交互设计
